Projetos em Ciência de dados

Desenvolvimento do um Painel Gerencial para Negócio com o Streamlit da empresa Curry Company

Nesse projeto os conceitos de programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio junto com as ferramentas de desenvolvimento web como o Streamlit e o Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na Índia.

O resultado do projeto foi um painel hospedado em um ambiente em Cloud disponibilizado por um link web. O painel pode ser acessado por qualquer dispositivo conectado a internet.
